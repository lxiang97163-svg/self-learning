# 盘中观察：板块够强 + 分时确认（Agent 维护稿）

---

## 1. 使用时机（与主线关系）

- **早盘竞价**：先看「第一优先 / 第二优先 / 低吸」是否给出可执行条件。
- **盘中观察**：仅当上面**没有舒服买点**（或只能极小仓）时启用；**禁止**与第一/二优先 **重复重仓**；**禁止**盘中临时新加未写入速查的代码。

---

## 2. 板块够强（原 B 层，速查里称「板块够强」）

以下 **三条至少满足两条**，且上证 **未有效跌破** 速查中的关键支撑（与当日复盘表「关键点位-支撑」一致，如 3794.68）：

1. 该题材在行情软件中的 **概念板块** 当日涨幅 **≥ +0.8%**（速查若写更高阈值以速查为准），或该板块处于两市 **概念涨幅前 1/3**（软件固定用一种）。
2. 该观察包 **5 只** 中 **≥ 2 只** 现价较昨收 **≥ +2.0%**。
3. 包内 **≥ 1 只** 涨停或涨幅 **≥ +9.5%**，且 **另有 ≥ 1 只** 涨幅 **≥ +5%**。

若三条里 **≤ 1 条** 成立 → **该包当日不开仓**。

可在 10:00、11:00、13:30 各扫一次，规则不变。

---

## 3. 分时确认（原 C 层，速查里称「分时确认」）

在 **板块够强** 前提下，拟买入的那一只须在 **下单前** 满足 **下列「顺势」或「弱转强」二选一**。

### 顺势（平开～高开走强）

- 最近 **5 根 1 分钟 K 线** 收盘价 **≥ 分时均价**（或 **≥ VWAP**），且当前价 **≥ 开盘价约 +1.0%**；
- 当日成交额或量比：**量比 ≥ 1.2**（或成交额 ≥ 近 5 日同期均额 × 0.25）。

### 弱转强（低开/杀跌后修复）

- 今开较昨收 **≤ -3%**，但现价已 **较昨收翻红**；
- 现价 **≥ 9:35 前分时高点**；
- 翻红后 **连续 3 根 1 分钟 K 线** 收盘不创新低。

**不买**（任一条）：大盘分时跌破关键支撑且 15 分钟内未收回；个股跌破日内低点约 **-2%**；同题材 **≥ 2 只** 同时炸板且跟风跳水。

---

## 4. 观察包顺位（固定含义）

| 顺位 | 含义 |
|:----:|------|
| 1 | 龙头 / 最高身位 |
| 2 | 龙二 / 补涨龙 |
| 3 | 中军 / 大成交额 |
| 4 | 弹性 / 小票 |
| 5 | 跟风 / 首板备选 |

---

## 5. 上车顺序

1. 若第一/二优先已按竞价**满仓执行** → 一般不再用同题材盘中观察**重复加仓**（除非速查写明例外）。
2. **板块不够强** → 该包**不买**。
3. **板块够强**：从顺位 **1→2→3→4→5** 检查，**第一只**满足「分时确认」的 **开仓**，**同一包当天只选一只**。若 **1～3** 均不满足分时确认 → **默认不追 4～5**（除非速查写明「极强市开放 4～5」）。
4. 单笔 ≤ **机动仓** 且 ≤ **单票上限**，与低吸合并算总仓。

---

## 6. 与低吸候选的关系

- 低吸偏 **竞价/开盘**；盘中观察偏 **竞价不强、盘中走强**。
- 同一只票两处都出现 → **不重复重仓**；分时以本节「分时确认」为准。

---

- 阈值（0.8%、+2%、量比 1.2 等）变更时：更新本文件，并在用户可见速查里用**同一套白话**写清，勿写脚本名。
- Agent 生成速查时：须含 **2～3 个** 支线观察包，每包 **5 只**，不得留空。
