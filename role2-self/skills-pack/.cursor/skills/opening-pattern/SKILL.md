# 开盘八法盘型判断

## 触发条件

用户问「今天什么盘型」「今天开盘怎么样」「早盘盘型」「开盘八法」等。

---

## 核心原则：破/量/比盘，缺一不可

> 一高盘：**破量比**；三高盘：**破比量**；三低盘：**量破比**；平移盘：**比量**

**每次判断必须用数据回答这三个问题：**
1. **破**：今日有没有破前高/压力区/均线/前日高点？
2. **量**：成交额相对昨日是爆量/增量/缩量/量窒息？
3. **比盘**：前一日是什么盘？主流板块今日是否续动？

---

## 执行流程

### Step 1：运行数据采集脚本（一次性拉取全部数据）

**直接执行** `pipeline/opening_pattern_data.py`，该脚本一次性输出以下6个模块：

| 模块 | 内容 | 对应判断维度 |
|------|------|-------------|
| 【1】实时行情 | 今开、昨收、现价、跳空幅度、成交额 | 开盘框架 |
| 【2】前三根5分钟K线 | 第1/2/3根收盘价 + 自动判断走法 | **盘型定性（最关键）** |
| 【3】近期日K线 | 近15日日K + 量能对比 + 位阶判断 | **量** + 位阶 |
| 【4】概念板块TOP10 | 今日涨幅前10概念板块 | **比盘** |
| 【5】行业板块TOP10 | 今日涨幅前10行业板块 | **比盘** |
| 【6】破 | 是否突破昨日最高/近5日波段高 | **破** |

```
python pipeline/opening_pattern_data.py
```

**不允许省略此步骤。不允许用新闻搜索替代数据接口。**

**数据源自动降级（脚本已实现，以输出中的「数据源」标注为准）：**

| 模块 | 优先 | 备用 |
|------|------|------|
| 【1】实时三指数 | 东财 `push2.eastmoney.com` | `push2delay.eastmoney.com`；仍失败则上证用 **腾讯** `qt.gtimg.cn?q=sh000001` |
| 【2】5分钟K | 东财 `push2his` | **新浪** `CN_MarketData.getKLineData`（scale=5）；脚本第7字段为成交量时，同时段量比按成交量口径 |
| 【3】日K / 盘中量能 | 东财日K + 5分钟额 | **新浪**日K（scale=240）；盘中同时段：东财5分钟失败则换新浪5分钟；新浪日K不含当日收盘时，脚本会用当日5分钟+腾讯现价**补一行今日**便于「破/位阶」 |
| 【4】【5】板块TOP10 | 东财 `clist` | **`push2delay`** 同路径 |
| 【6】压力与「破」 | 东财日K | **新浪**长日K + 必要时补今日 |

---

### Step 2：读取脚本输出，逐项提取判断依据

从脚本输出中提取：

**开盘框架**（来自模块1）：
- 跳空 > +0.5% → 一高系开盘
- 跳空 < -0.5% → 一低系开盘
- 介于 ±0.5%，今开 > 昨收 → 平移下开盘（近平开略高）
- 介于 ±0.5%，今开 < 昨收 → 平移上开盘（近平开略低）

**前三根5分钟走法**（来自模块2，脚本已自动判断）：
- 收盘逐级走高 → 三高盘
- 前两根走高第三根平 → 二高盘
- 收盘逐级走低 → 三低盘

**量能**（来自模块3）：
- **收盘后**：脚本自动比较今日全天 vs 昨日全天成交额
- **盘中**：脚本自动拉今日和昨日5分钟K线，按**同时段累计成交额**对比，禁止用全天数据代替盘中判断

| 量能状态 | 比值 |
|----------|------|
| 爆量 | > 1.5x |
| 增量 | 1.1~1.5x |
| 平量 | 0.9~1.1x |
| 缩量 | 0.6~0.9x |
| 量窒息 | < 0.6x |

**比盘**（来自模块4、5）：
- 对比昨日领涨板块，判断主流板块是否续动
- 主板不变则上攻不止；换板块 = 气势虚弱的补动

**破**（来自模块6）：
- 脚本已自动判断是否破昨日最高、近5日波段高

**位阶**（来自模块3）：
- 脚本已自动计算近5日高低档位置百分比

## 定性用语规范（必须使用笔记原文术语）

| 最终盘型 | 标准名称 |
|----------|----------|
| 三根5分钟收盘逐级走高 | **三高盘（帝王盘）** |
| 前两根走高第三根平 | **二高盘（辅臣盘）** |
| 高开收最高/次高 | **一高破高先锋盘** |
| 高开震荡未破压力 | **一高压力盘** |
| 高开收最低/次低 | **飘盘** |
| 低开向上弹收红 | **一低承接盘** |
| 低开继续下杀 | **一低挫头盘** |
| 三根5分钟收盘逐级走低 | **三低盘（冥王盘）** |

**禁止**：
- 跳过5分钟K线直接用新闻定盘型
- 只看跳空幅度就下最终结论
- 缺少量能/比盘/破任何一个维度的数据就输出结论

## 各盘型操作建议速查

### 三高盘（帝王盘）
- **核心**：破比量。先看有没有「破」，再比前日盘，最后看量
- **操作**：低档可跟进，重点看是否破前高/压力区；高档防爆量赶顶
- **量能**：量缩比量增还要好；爆量在高档 = 散户追价，危险
- **风险**：高档三高前后低开 = 孤王盘，是最高点暗示
- **次日**：三高后常跟一高（先锋），震荡拉回皆买点

### 二高盘（辅臣盘）
- **核心**：量比破。要量增，主流板块要续动
- **操作**：多方整理，量增续攻；主流板块不变则上攻不止
- **量能**：量增为好，爆量要小心（功高震主）
- **风险**：二高后三低，一定不要追高
- **次日**：爆量二高次日必整理

### 一高破高先锋盘
- **核心**：破量比。「破」是第一位
- **操作**：可跟进，重点看是否破前高/压力区
- **量能**：爆量跳空高开为典型；缩量开盘只宜低档
- **风险**：高档爆量一高瞬间拉黑 = 大压力盘，不追
- **次日**：一高之后喜帝王（三高），辅臣在后补价

### 一高压力盘
- **操作**：当日不追高，等回落确认支撑
- **量能**：量缩量增均是压力，高档尤忌
- **风险**：一点高盘不胜寒（前后都低开）
- **次日**：次日开低防空方转换

### 飘盘
- **判断要点**：必须收在最低/次低才能叫飘
- **操作**：当日不买，等探底确认
- **风险**：飘后开低不能买（除非平移上）
- **次日**：飘盘当日收低，次日不能低开

### 一低承接盘
- **操作**：低档有承接，低开后往上弹可跟进
- **量能**：低档爆量一低 = 背离，第二天开高确认
- **风险**：高档一低 = 挫头盘，不参与
- **次日**：一低向上走拉尾盘，次日即使低开当天还有创高机会

### 三低盘（冥王盘）
- **核心**：量破比。先看量，再看破，最后比盘
- **操作**：低档量窒息三低 = 赶底，等霸王举鼎机会
- **量能**：低档喜爆量或量窒息；高档增量三低 = 真杀盘
- **风险**：高档三低 = 大杀盘，不参与
- **次日**：三低后只能高开或平移上；再开一低 = 续低型

### 平移上
- **核心**：比量。缩量平移上当一低盘用
- **操作**：量缩整理，等突破确认
- **风险**：高档量窒平移上 = 做头风险

### 平移下
- **核心**：比量。增量平移下当一高盘用
- **操作**：整理盘，比量比盘比主流板块
- **风险**：高档量窒平移下 = 量能退潮，防做头

---

## 注意事项

1. **9:30 前**：只能定「一高/一低/平移」框架
2. **9:30–9:45**：可定具体盘型，**必须看5分钟K线**
3. **飘盘判断**：必须收在最低/次低才能叫飘
4. **量能对比**：相对昨日同时段，不是绝对值；盘中数据注意时段差异
5. **位阶判断**：高档/低档影响盘型含义
6. **不确定时**：明确说明「当前信息只能定到[X层]，[Y层]需要更多数据」
