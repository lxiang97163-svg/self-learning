# AI创作者学习体系 · 快速参考

## 每日流程（约15-30分钟）

```
1. 打开内容源 → 2. 快速浏览今日更新 → 3. 复制模板填写「今日摄入」
      ↓
4. 提炼「今日整合」（跨人共性/冲突）→ 5. 列「可执行清单」→ 6. 沉淀1-2个词条
```

## 内容源打开顺序

| 顺序 | 平台 | 操作 |
|------|------|------|
| 1 | 微信 | 订阅号消息 → 数字生命卡兹克、艾逗笔、花叔等 |
| 2 | 邮箱 | Newsletter：Dan Koe、AI-Nate 等 |
| 3 | Twitter/X | 关注列表 → 按兴趣快速扫 |
| 4 | RSS | Inoreader / Feedly → 公众号RSS、博客 |

## 模板路径

- 每日模板：`templates/每日学习_模板.md`
- 每日输出：`daily/每日学习_YYYYMMDD.md`
- 词条库：`knowledge/` （自建词条 md）

## 整合三问

1. **共性**：不同人今天都提到什么？能抽象成规律吗？
2. **冲突**：谁和谁观点对立？我更信谁？为什么？
3. **可做**：今天能立刻做的一件小事是什么？
