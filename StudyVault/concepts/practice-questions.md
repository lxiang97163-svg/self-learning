# 练习题库 — frontend-design

#frontend-design

---

## 设计思维

1. 使用 frontend-design skill 时，开始写代码前必须先回答哪 4 个问题？
2. "Tone" 应该是模糊的方向还是具体的极端？举例说明区别。
3. "Differentiation" 要求你回答什么问题？为什么这是最重要的问题？
4. 极简主义（minimalist）设计的"优雅"体现在哪里？

## 排版

5. 为什么不能使用 Inter 字体？
6. 展示字体（display font）和正文字体（body font）配对时应制造什么效果？
7. 同时使用超过 2 种字体会有什么问题？
8. `font-display: swap` 的作用是什么？

## 色彩与主题

9. "主色调主导 + 1-2 个 accent"胜过"5-6 色均匀分布"的原因是什么？
10. 为什么说紫色渐变+白底是典型反模式？
11. CSS 变量在色彩系统中解决了什么问题？
12. 深色主题不一定比浅色高级——判断用哪个的依据是什么？

## 动效

13. "交错出现（staggered reveal）"是什么技术？用什么 CSS 属性实现？
14. 动效性能中，`width` 和 `transform` 哪个更安全？为什么？
15. 动效时长的"甜点区间"是多少毫秒？
16. Motion 库适合在什么项目/场景中使用？

## 空间构图

17. 为什么说"所有东西居中"是平庸的选择？
18. "等间距"布局有什么问题？正确做法是什么？
19. "打破网格"应该用于哪些区域？全页打破网格会怎样？
20. 极度负空间 vs 极致密排——什么情况选哪个？

## 背景与视觉细节

21. 给纯色背景加一层 opacity 极低的噪点，会产生什么效果？
22. 渐变网格（gradient mesh）用什么 CSS 属性实现？
23. 背景效果的"不透明度"建议保持在什么范围，为什么？
24. 自定义光标适合什么类型的页面？功能性 App 适合吗？

## 反模式

25. 判断一个设计是否 "AI slop" 可以用哪 3 个自检问题？
26. 列举 3 种布局级反模式。
27. 避开紫色渐变是否意味着必须选某个特定颜色？为什么？
28. 每次设计使用不同字体的原因是什么？
