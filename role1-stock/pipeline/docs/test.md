## 附录｜结论摘要（≤10 行）

1. 主逻辑永远先走 **速查**：风标 → 电力/锂电验证链 → 如果则 → 破位降仓。  
2. **方向扩展**解决「除速查票池外，还有哪些分支可能在抢资金」。  
3. P0～P2 与速查一致；P3 仅在 P0、P1 不满足或明确轮动时启用。  
4. 本文件为 **test 样例**，数值与标的仅供版式演示，**下单前以当日 `速查_YYYY-MM-DD.md` 与实时行情为准**。
