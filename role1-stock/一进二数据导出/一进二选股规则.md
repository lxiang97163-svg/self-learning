# 一进二选股规则（书面定稿）
> 生成方式：由 `构建一进二选股规则.py` 拉取**严格首板∪严格二板联合池**与 9:25 竞价数据，基于「参考样本包络 + 原型相似度排序」直接落盘；**未**引用 `一进二_grid_search.py` / `一进二_9点28选股.py` 的逻辑。
## 一、推测过程（摘要）
1. **池**：T-1 日 `limit_list_d` 中严格首板（`limit_times==1`、T-2 未涨停）∪ 严格二板（`limit_times==2`）∪ 三板（`limit_times==3`），均限沪深主板、非 ST；竞价与名次在联合池内计算。
2. **次日竞价**：仅使用 T 日 `stk_auction` 在池内标的上的涨幅、成交额、换手率；在池内计算涨幅排名、换手排名、成交额排名。
3. **参考样本（全部为锚定）**：`2026-04-10` **华远控股 `600743.SH`**；`2026-04-10` **和胜股份 `002824.SZ`**；`2026-04-13` **康盛股份 `002418.SZ`**；`2026-04-15` **哈药股份 `600664.SH`**；`2026-04-17` **均瑶健康 `605388.SH`**；`2026-04-20` **合力泰 `002217.SZ`**。
4. **量化推断**：对参考样本的各字段取保守包络，并按可读粒度向外取整。例如涨幅区间取参考样本最小/最大值后按 0.1% 外扩，成交额与换手率按 0.01 外扩；三类排名不再使用跨日绝对名次，而改用**池内名次占比**包络；命中结果再按参考样本原型的相似度排序。
5. **样本内回放日**：对 `20260410`、`20260413`、`20260415`、`20260417`、`20260420` 做样本内回放，检查同一套硬条件是否均能产生命中，并用收盘相对竞价近似回看强弱；**这不是独立样本外验证**。
6. **反例验证日**：`20260414`、`20260416` 为已知无合适标的的交易日，规则需确保这些日期**命中 0 只**；若包络参数仍有命中，脚本会自动贪心收紧参数（每轮选最优单步），直到反例全部清零且锚定不丢失。
7. **排除**：不采用流通市值 50~80 亿硬区间；本版也不再做大规模参数穷举。
8. **三维权**：`pct_rank_ratio_max`、`turn_rank_ratio_max`、`amt_rank_ratio_max` 为有限小数时，表示标的须同时满足「竞价涨幅名次占比 ≤ 该值」「换手名次占比 ≤ 该值」「竞价成交额名次占比 ≤ 该值」（名次占比 = 当日池内名次 / 当日池大小）。
## 二、定稿规则（可执行）
### 2.1 标的池（T-1）
- **严格首板**：T-1 日涨停且 `limit_times=1`，T-2 未涨停，沪深主板，非 ST。
- **严格二板**：T-1 日涨停且 `limit_times=2`，沪深主板，非 ST。
- **三板**：T-1 日涨停且 `limit_times=3`，沪深主板，非 ST。
- 三路结果按 `ts_code` 去重取**并集**；竞价与名次均在该联合池内计算。
- 剔除：ST/*ST、创业板/科创板/北交所。

### 2.2 次日竞价过滤（T 日 9:25）
- 竞价涨幅：3.3% ≤ 涨幅 ≤ 9.99%。
- 竞价成交额 ≥ 0.33 亿元。
- 竞价换手率：0.5% ≤ 换手率 ≤ 0.93%。
- 竞价涨幅在**当日首板池内**排名：前 18%（按池内名次占比）。
- 竞价换手率在**当日首板池内**排名：前 25%（按池内名次占比）。
- 竞价成交额在**当日首板池内**排名：前 44%（按池内名次占比）。
- 近 5 日涨幅 ≤ 30%。
- 近 10 日涨幅 ≤ 33%。
- 新高约束：是（竞价价须不低于近 20 或近 60 日最高价）。
- T-1 开板次数 ≤ 5 次（来自 `limit_list_d`，全锚定均有值时生效）。

### 2.3 输出
- 同时满足以上条件的全部股票即为「计算机选」结果；结果已按**原型相似度**由高到低排序。
- 若需压到 1~2 只，优先取原型距离最小者；原型距离越小，越接近参考样本的整体画像。
