## 核心知识速记

### 三条铁律
```
铁律1: 断板次日=情绪0分=强制空仓
铁律2: 越涨越不加仓（禁止正金字塔追涨）
铁律3: 趋势票建仓次日没利润电=认错出局
```

### 三一速记
```
操作三一 = 进额第一 + 换手第一 + 涨幅第一（竞价阶段）
理念三一 = 第一（强中选强）+ 唯一（辨识度）+ 专一（一击不重）
危险三一 = 分歧期/退潮期 + 盘子太小 → 被反噬
```

### 炸板六原则速记
```
1.时间（后炸板>先炸板）
2.封单（炸前封单大=强）
3.性质（被动>主动）
4.放量（慢>快）
5.盘口（主力净额向上）
6.回风（快且大=强）
```

### 情绪节点打分
```
启动=3 → 上升=2 → 高潮=1 → 分歧=1 → 退潮=0 → 断板次日=0
```
