# 06 专项技术

## 1. 个股顺位定义（市场地位排序）

### 1.1 顺位表

| 优先级 | 地位名称 | 定义 | 操作价值 |
|--------|----------|------|---------|
| 1（最高）| **妖股** | 完成两次爆量高开转移制的票 | 资金有记忆，容易有二波；最优 |
| 2 | **市场龙头** | 当前市场情绪代表性最强的票 | 引领情绪，是情绪风标首选 |
| 3 | **题材龙头** | 某题材内连板数最高/最强的票 | 题材主攻方向，买点最多 |
| 4 | **核心票/和星谷** | 近期可引导市场情绪的高辨识度标的 | 情绪有联动，做买卖点有效 |
| 5 | **人气中军** | 日成交额30-80亿的大盘趋势票 | 适合大资金，分时弹性相对低 |
| 6 | **趋势票** | 缓慢均线向上，成交量温和放大 | 慢牛，适合做T和持仓 |
| 7 | **前排跟风** | 题材内跟随龙头上涨的二、三线标的 | 赔率低，谨慎操作 |
| 8（最低）| **后排砸毛** | 题材高潮时才会跟涨的低辨识度票 | 分化来临第一个爆；不碰 |

### 1.2 操作优先级规则

```
IF 在多只票中选择:
    优先级：妖股 > 市场龙头 > 题材龙头 > 核心票 > 人气中军
    
    绝对禁止：在有龙头的前提下，去做砸毛票
    
IF 找不到龙头买点:
    退而求其次：核心票 → 趋势票
    不降级到跟风/砸毛
```

### 1.3 妖股认定标准（精确定义）

```
条件1：在连板运行中经历了两次爆量高开转移制
  第一次爆量高开转移制 → 市场龙头候选
  第二次爆量高开转移制 → 正式成为妖股

条件2（替代路径）：经历一次爆量高开转移制 + 一次三一（市场级别）
  → 两者各一次即可认定

判断口诀:
  "一波爆量，龙头候选；两波爆量，资金记忆；带量二波，妖股生涯"

已验证妖股案例（历史参考）:
  - 圣龙股份：两次爆量高开转移 → 多次产生二波
  - 爱金宫：同样逻辑，资金记忆极强
  - 三伯朔：第二次爆量量能>第一次 → 极强妖股
```

## 2. 盘口分析技术

### 2.1 盘口一致性判断（开盘前3笔法）

```
判断逻辑：
  开盘后前3笔成交（每笔约3秒一回报）
  
  IF 连续3笔成交价格上行且量能增加:
    → 买方主导 → 持仓或加仓
    
  IF 连续3笔成交价格下行:
    → 卖方主导 → 等待止跌或放弃
    
  IF 价格震荡，量能下降:
    → 无方向 → 等5分K方向确认
```

### 2.2 主力净额方向判断

```
主力净额（大单净流向）:
  持续向上 = 主力在吸筹/维护筹码 = 看多信号
  急速下行 = 主力出货/减仓 = 看空信号
  横盘震荡 = 筹码博弈 = 等待方向

配合分时图使用:
  量价同向 + 主力净额向上 = 最强买入信号
  量价背离 + 主力净额下行 = 减仓/止损信号
```

### 2.3 封板强度判断

```
封板量:
  极强：封单 > 流通市值5% （锁定大部分流通盘）
  强：封单 > 1亿（绝对金额够大）
  一般：封单 < 5000万（容易被打开）

封板风险识别:
  IF 封单金额 < 炸板时成交量 → 封不住，高风险
  IF 封板后量能放大（5分K量柱增高）→ 有人刻意打板，风险上升
  IF 封板后量能缩小 → 筹码稳定，封板稳健
```

## 3. 复盘六步流程

### 3.1 每日复盘标准流程

```
Step 1: 指数复盘
  - 今日关键点位守住了吗？
  - 出现了哪种K线信号（预祝晴天/复堂/仙人指路）？
  - 明日关键点位是多少？
  
Step 2: 情绪复盘
  - 今日情绪风标表现？
  - 封板数量、跌停数量、红绿盘比例
  - 情绪节点在哪个阶段？
  
Step 3: 题材/板块复盘
  - 今日主线是什么？
  - 板块内T队梯队情况（9-4-3-1）？
  - 题材运行到哪个节点？
  - 板块内部分支各自表现？

Step 4: 个股复盘（同花顺660）
  - 涨幅前列票的图形和模式
  - 换手率、主力净额排序
  - 五日涨幅最高的是哪些票（情绪强度参考）
  - 尾盘进价抢筹情况（3:00看进价进额排序）
  
Step 5: 龙虎榜复盘
  - 机构净买入的是什么票？
  - 游资（知名席位）在哪个板块？
  - 机构净卖出的是什么票？（出货警示）
  
Step 6: 消息面/新闻
  - 盘后重要消息（政策/公告）
  - 韭菜公社等平台的题材梳理
  - 明日可能有催化剂的题材
```

### 3.2 龙虎榜上榜条件（主板）

```
满足以下任一条件即上榜：
1. 当日价格涨跌幅偏离值超过7%
2. 当日换手率≥20%
3. 价格涨幅≥15%
4. 连续3个交易日累计涨幅偏离值≥20%
```

## 4. 趋势票做T完整体系

### 4.1 做T五要件

```
要件1: 市场环境（最重要）
  - 指数趋势 + 题材运行阶段 + 情绪节点
  - IF 退潮期 → 做反T（卖高买低，不做正T）
  - IF 上升期 → 可做正T（低买高卖）

要件2: 支撑位和压力位
  - 六种DC买点模式（以下4种最常用）：
    a. 绿盘DC（水下缩量+题材主流时DC）
    b. 5分K支撑DC（精确找到5分K支撑）
    c. MA5/MA20支撑DC
    d. 前高/前低作为支撑DC

要件3: 分时量价背离
  - 顶背离（高位放量+价格不创新高）→ 减仓
  - 底背离（低位缩量+价格不创新低）→ 买入

要件4: 盘口信号
  - 盘口衰竭（大卖单增多，买单减少）→ S点
  - 盘口攻击（大买单连续出现）→ B点

要件5: 等仓原则
  - 买T仓量 ≤ 当前底仓量
  - 当日必须等量卖出T仓
  - 绝对禁止T加仓变成加仓
```

### 4.2 趋势票建仓逻辑

```
建仓成功条件:
  次日（建仓后第二天）必须做出利润电
  
  IF 建仓次日不高开 AND 无法做出利润电:
    → 认错出局（无条件，不磨蹭）
    → "火箭发射失败，不等待，立刻离开"
    
建仓失败后的禁区:
  - 不等待第3天
  - 不等支撑位
  - 不抱希望它会反弹
  - 直接认错，止损出局

建仓成功后的持仓逻辑:
  有利润电（>5%浮盈）→ 开始做T
  做T把成本降到安全位 → 长期持有观察
  每次反弹都做减仓T → 成本越来越低
```

### 4.3 绿盘DC买点（最常用T点）

```
触发条件:
  1. 题材处于上升期（不是退潮期）
  2. 个股处于水下（当日跌幅）
  3. 量能极度缩量（< 前日量能的50%）
  4. 大单净额持续向上（主力没跑）
  5. 触及MA5/MA20或前期支撑价格

确认信号:
  - 小WD形态（二次踩脚，第二脚不低于第一脚）
  - 5分K在支撑位附近出现缩量阳线

买入:
  - 下单在支撑价格±1分钱挂限价单
  - 接到了就接到了，没接到就放弃（不追）
```

## 5. 集合竞价分时图完整解读

### 5.1 四象限结构

```
         [上方 = 未成交挂单区]
          红色柱 = 买单排队（涨停价买单）
          绿色柱 = 卖单排队（涨停价卖单）
         [中线 = 9:20分时间节点]
         [下方 = 已成交区]
          红色 = 买方成交
          绿色 = 卖方成交
```

### 5.2 四种典型竞价分时形态

**形态A：一致抢筹（最强）**
```
上方：红色大柱（大量买单排涨停）
下方：红色小柱（成交少=买的人多，卖的人少）
含义：高度一致性抢筹，次日大概率强势
9:20后：若买单不减反增 → 极强
```

**形态B：抛压型**
```
上方：红绿混杂，绿色（卖单）逐渐增多
下方：绿色增大
含义：持仓者趁机兑现，抛压大
→ 跟随卖出，不追买
```

**形态C：凹槽型（主力试盘）**
```
9:15-9:20：红色大（买单多）
9:20-9:25：红色缩小（主力撤单）
9:25后：红色重新增大（主力重新买入）
含义：主力9:20撤单试探，9:25再入=最佳买点
→ 9:25进场
```

**形态D：撬跌停型**
```
上方：绿色大（跌停卖单多）
9:20后：绿色缩小（卖单撤走）
含义：抛压减少，可能撬板
→ 关注是否尾盘成交放量
```

### 5.3 竞价尾盘急速拉升

```
9:24分后成交量（下方）急速拉升:
  → 说明有资金在最后时刻大量抢筹
  → 强做多信号，9:30开盘大概率高开走强
  
配合判断:
  成交量拉升 + 封单增加 → 超强信号
  成交量拉升 + 封单不变 → 普通信号
  成交量拉升 + 封单减少 → 抢筹者在对抗抛压，有风险
```

## 6. 炸板六原则（完整版）

| 原则 | 评估维度 | 强信号 | 弱信号 |
|------|----------|--------|--------|
| 1 | 炸板时间 | 晚炸（10点后）| 早炸（开盘即炸）|
| 2 | 炸板前封单 | 封单>5亿 | 封单<1亿 |
| 3 | 炸板性质 | 被动（指数原因）| 主动（主力出货）|
| 4 | 放量速度 | 缓慢放量 | 光速放量（10秒放量>前日成交）|
| 5 | 盘口变化 | 主力净额向上 | 主力净额急速下行 |
| 6 | 回风时间/力度 | 快速回风（5-10分钟内）| 长时间回不来 |

**炸板量能判断技巧**：
```
关键公式：
  炸板时的分时量柱 vs 封板时的分时量柱
  
  IF 炸板量柱 < 封板量柱 → 相对强势，可低阶
  IF 炸板量柱 > 封板量柱 → 抛压超过买盘，高风险
  
  IF 炸板前总成交 + 炸板瞬间成交 ≈ 昨日总成交 → 已充分换手，不宜低阶
```

## 7. 短线精灵设置（推荐配置）

```
成交异动：
  ✅ 大笔买入
  ✅ 大笔卖出

走势异动：
  ✅ 涨幅突破（建议设置5%阈值）
  ✅ 涨停封满
  ✅ 打开涨停
  ❌ 跌停封满（可选，不必须）
  ❌ 打开跌停（同上）

急速类：
  ✅ 急速拉伸
  ✅ 猛烈打压
  ✅ 涨停大检
  ✅ 跌停大检（用于监控核心票是否崩盘）

撤单异动：
  ✅ 撤涨停买单（监控强跑信号）
  ❌ 其他撤单（杂音太多，不看）

注意：
  - 若跳动太频繁影响判断，删除"跌停封满"
  - 杂音越少越好，保留高价值信号即可
```
