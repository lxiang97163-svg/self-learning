# 07 龙头与龙周期

## 核心认知：龙头是市场情绪的载体

龙头不是基本面最好的票，而是**当下市场情绪的物化表达**。跟着龙头走，就是跟着最强的市场共识走。

---

## 1. 情绪节点闪电模型

### 1.1 龙周期的七个节点演化

```
[启动] → [发酵] → [高潮] → [分歧] → [分化] → [退潮] → [混沌]
                                                    ↓
                                          [下一轮启动]（新龙头出现）
```

**核心规律**：
- 每个节点转换都有**明确的市场信号**触发
- 退潮到下一轮启动之间，可能有**新旧龙头无缝切换**
- 情绪的死亡和新生在盘中是**同步发生**的（闪电模型）

### 1.2 闪电模型（无缝切换逻辑）

```
当日盘中的切换时刻:
  旧龙头：退潮/分歧/断板
  同时：新龙头（新题材/新个股）开始接力
  
观察点：当旧龙头开板/断板的同时，市场最强资金去了哪里？
  → 去的地方 = 下一个龙头题材/个股
  
时间特征：
  闪电切换：当天即完成
  滞后切换：1-3天后新龙头明确
```

### 1.3 龙头的生命周期（理想模型）

```
Day 1-2: 题材启动，多个一字板，龙头出现
Day 3-5: 龙头连续封板，T队梯队扩大（发酵期）
Day 6-8: 龙头冲高，封单达到历史最大（高潮期）
Day 9  : 龙头高开后出现分歧（首次开板/缩量）
Day 10 : 龙头断板（退潮信号）
Day 11+: 情绪清零，等待下一轮
```

## 2. 龙头的成长路径

### 2.1 普通票→龙头的必经阶段

```
路径A（更常见）:
  ① 首日出现大量资金介入（进价三一）
  ② 连续封板（形成连板龙头）
  ③ 经历第一次爆量高开转移制 → 成为市场龙头候选
  ④ 经历第二次爆量高开转移制 → 正式成为"妖股"，有资金记忆

路径B（弱转强）:
  ① 先成为空头风标（跌停）
  ② 次日竞价绿三一（在下跌股中换手最高）
  ③ 主力资金确认反核 → 弱转强龙头诞生
```

### 2.2 龙头排位竞争逻辑

```
同一题材内的排位竞争:
  判断维度：
    a. 连板数（越多越强）
    b. 封单大小（越大越强）
    c. 竞价表现（进额/换手/涨幅）
    d. 盘子大小（适中最优，<10亿太小，>100亿太大）
    
  排位更迭信号:
    旧龙头断板/分歧 → 次强票接力（T队中的第二名升级）
    新题材爆发 → 龙头切换到新题材
```

## 3. 龙周期复刻逻辑

### 3.1 周期复刻的本质

```
市场特征：相似的情绪节点 + 相似的题材结构 → 大概率相似的走势
  
应用：
  某段历史中题材A的运行节点 → 被当前市场题材B复刻
  复刻越精确，操作胜率越高
  
复刻判断步骤:
  Step 1: 找到当前市场最强题材的情绪节点位置
  Step 2: 在历史中找"启动→高潮"节点结构相似的案例
  Step 3: 对比当前龙头的板数/封单/题材热度是否接近历史案例
  Step 4: 如果接近，可参考历史案例的后续走法
```

### 3.2 典型案例：固态电池vs半导体的复刻

```
时间线对比:
  固态电池 9月8号 高潮期（市场龙头出现）
  半导体   9月25号 高潮期（情绪完全复刻）
  
  对比要点:
  - 启动节点时间差：约17天
  - 情绪运行结构：完全一致
  - 龙头行为模式：相同（三一→爆量→妖股路径）
  
应用价值:
  当你看到当前题材进入高潮期
  → 对照17天前的题材峰值，判断当前有多少剩余空间
  → 参考历史龙头的后续走法，设定目标和止损
```

## 4. 核心龙头的具体案例解析

### 4.1 韩武基科技牛启动（8月8号）

```
背景：8月8日英威克异动，算力/科技板块启动
信号：英威克进价三一，封单破历史
意义：科技行情的真正起点（回头看是起点，当时需实时判断）

判断方法:
  IF 某票进价金额达历史新高（亿级别）
  AND 题材是当前市场唯一热点
  AND 指数环境支持（高开/强势）
  → 确认为情绪启动，猛上仓位
```

### 4.2 龙头切换时的操作节点

**Case: 从一个龙头到另一个龙头**

```
时序图:
  T日: 旧龙头A断板（退潮信号）
       同时/次日: 新题材B开始启动
       
操作:
  T日收盘: A减仓至轻仓或清仓
  T+1日进价: 观察B进额/换手是否三一
  T+1日9:30: IF B满足条件 → 买入B
              IF B未确认 → 空仓观察一天

关键判断：新龙头的题材是否有"唯一性"？
  有唯一性（市场绕不开）→ 可做
  无唯一性（模糊/多选）→ 等主线确认
```

## 5. BS点系统（买卖点体系）

### 5.1 B点（买入点）层次

```
B1（最优买点）:
  - 题材启动第一天的进价三一
  - 先人指路第一日高开
  - 前排炸板低阶（被动炸板后的第一次回踩）
  
B2（次优买点）:
  - 爆量高开转移制的次日
  - B1之后价格回调至支撑位
  
B3（补仓点）:
  - 持仓后的下一个模式内买点
  - 已有底仓前提下的追加
  
B4及以后:
  - 买点越往后，赔率越低
  - 必须控制仓位，不可全力出击
```

### 5.2 S点（卖出点）层次

```
S1（最强卖出）:
  - 高档爆量（当日成交超过前期最大量）
  - 量加背离（量能达新高但价格不新高）
  
S2（正常卖出）:
  - 龙头断板（情绪清零，减仓信号）
  - 盘口衰竭（大卖单持续出现）
  
S3（防守卖出）:
  - 题材出现分化（T队梯队开始断层）
  - 指数关键点位被破（系统性风险）
  
S4（止损）:
  - 趋势票建仓次日未做出利润电
  - 短线票破5分K关键支撑
```

## 6. 经验库使用方法

### 6.1 经验库的作用

```
目的：记录同一个票在不同板数时的胜率统计
格式：股票名 | 板数 | 买入结果（WIN/LOSE/FLAT）| 买入条件

使用方法:
  1. 看目标票的历史胜率
  2. 在当前板数下，该票的买入胜率是多少？
  3. 高胜率（>60%）→ 可正常参与
  4. 低胜率（<40%）→ 轻仓或放弃
  
记录维度:
  - 命中：正确选出并获利
  - 冷启动：新票首次参与，无历史数据
  - 题材崩盘：题材崩溃导致亏损
  - 正确过滤：识别出风险，正确不参与
  - 阈值偏低：竞价阈值设置偏低导致的误入
```

### 6.2 胜率累积方法

```
每日操作后记录：
  票名 + 板数 + 当日操作 + 结果

每周回顾：
  哪些模式/哪些票/哪个板数段胜率最高？
  
调整策略：
  IF 某票在3板以上历史胜率 < 40% → 3板后不参与
  IF 某题材在退潮期买入历史亏损率 > 80% → 退潮期永远不买该题材
```

## 7. 龙周期决策树

```
每日盘中龙周期判断流程:

Step 1: 当前市场龙头是谁？
  → 最高板股 = 市场龙头候选
  → 封单最大 + 题材辨识度最高 → 确认龙头

Step 2: 龙头在什么情绪节点？
  启动/发酵 → 买入/持仓
  高潮 → 轻仓，准备减
  分歧/分化 → 控仓，不加
  退潮/断板 → 立刻减仓或清仓

Step 3: 是否有新龙头正在形成？
  IF 旧龙头退潮 AND 有新题材进额异动:
    → 关注新题材进价三一
    → 等9:30后确认新龙头

Step 4: 新旧龙头切换后的操作
  旧龙头：持仓减至0-1层
  新龙头：满足三一则新建3层仓
```
