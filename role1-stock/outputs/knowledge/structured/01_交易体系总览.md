# 01 交易体系总览

## 1. 核心交易框架

### 1.1 四维分析维度（大局观骨架）

| 维度 | 权重分 | 对应仓位层 | 评分逻辑 |
|------|--------|------------|----------|
| 指数 | 4分 | 4层仓 | 高开供给盘=4；震荡=3；低开但强支撑=2；破位=1；空头排列=0 |
| 情绪 | 3分 | 3层仓 | 启动期=3；上升期=2；高潮/分歧=1；退潮/断板次日=0 |
| 题材 | 2分 | 2层仓 | 主流+蓄势=2；主流轮动=1；杂乱无主线=0 |
| 个股 | 1分 | 1层仓 | 满足模式+量价良好=1；不满足=0 |

**总分→仓位转换**：10分=满仓，实际以层数计（1层≈10%总资金）

## 2. 操作模式全图谱

### 2.1 强更强模式（进攻性买点）

| 模式 | 触发条件 | 核心逻辑 |
|------|----------|----------|
| **三一模式** | 集合竞价：进额第一 AND 换手第一 AND 涨幅第一 | 抢筹最激烈=资金一致性最高=次日仍强 |
| **半路模式** | 盘中出现量能突破+5分K站上关键价格 | 资金中途介入，仍有空间 |
| **9431板块选股** | 板块内九板→四板→三板→一板梯队齐整 | 梯队完整=情绪持续性强 |

### 2.2 弱转强模式（反转性买点）

| 模式 | 触发条件 | 注意事项 |
|------|----------|----------|
| **先人指路** | 昨日长上影线+今日带量高开 | 低开极拉视为修复，同样有效 |
| **断板反包** | 昨日断板（未封住涨停）+今日集合竞价高开 | 需题材龙头同方向配合 |
| **炸板低阶（前排）** | 被动炸板（指数原因）+炸板晚+回风快 | 主动炸板高位爆量=出货，不碰 |
| **爆量高开转移制** | 前日爆量+次日高开+分时继续上行 | 是妖股形成的必经阶段，第N次买点赔率递减 |
| **连板弱转强** | 空头风标反核+进价绿三一 | 结合情绪节点，分歧期出现需谨慎 |
| **一红定江山** | 长期横盘后跳空突破前115日最高价 | 量能必须配合，突破当日收盘不回补 |
| **空头风标反核** | 最大跌停票进价三一+情绪启动节点 | 绿三一=空头风标反核的典型信号 |

## 3. 股票确定性五要件

```
确定性 = 模式 × 信号 × 题材 × 量价 × 盘口

五要件缺一不可：
1. 模式   — 有明确的操作模式匹配（强更强 or 弱转强）
2. 信号   — K线/均线/竞价出现明确方向信号
3. 题材   — 所在题材是当日主流或有主流联动
4. 量价   — 量价配合良好（量能不爆亮，价格有支撑）
5. 盘口   — 开盘前3笔一致偏多
```

## 4. 如果-则执行清单（15条操作规则）

### 4.1 指数判断（4条）

```
IF 指数高开 AND 站上关键点位 → 可主动进攻，仓位给足
IF 指数低开 AND 未破前低支撑 → 轻仓观察，等待盘中机会
IF 指数破关键支撑（如3031/2984等） → 清仓或空仓，等信号
IF 指数出现空方仙人指路（跳空低开） → 当日禁止开新仓
```

### 4.2 情绪判断（3条）

```
IF 情绪风标（最高板/题材龙头）强势高开 → 情绪启动，可加仓
IF 情绪风标高开后断板 → 情绪即时归零，当日清仓观望
IF 龙头连续断板2日 → 退潮确认，等待下一个启动节点
```

### 4.3 题材判断（4条）

```
IF 同题材3+个股一字板 AND 封单≥1亿 → 题材确认，优先做龙头
IF 进价题材板块涨幅第一+换手第一+涨停加速第一 → 三重共振=主线确认
IF 板块T队梯队9-4-3-1整齐 → 题材持续性强，可做跟进
IF 龙头题材股发布利空公告 → 该题材当日谨慎，切换到次强题材
```

### 4.4 持仓判断（4条）

```
IF 持仓低开 AND 题材龙头正常 AND 指数正常 → 持仓不动，等回风
IF 持仓低开 AND 题材龙头也弱 → 减仓1/3，留底仓观察
IF 持仓低开 AND 指数+情绪双弱 → 开盘即清仓
IF 持仓冲高 AND 达到日内目标 → 主动减半仓，成本归零后持底仓
```

## 5. 交易哲学

### 5.1 先验主义
- 每日盘前写预案，设定IF-THEN操作路径
- 预案决策由理性驱动，盘中执行由纪律驱动
- "预案归预案，预期归预期，盘面说了算"

### 5.2 人为市场立法（核心认知）
- 市场不是被动发现的，是主力资金"创造"的叙事
- 题材=资金需要的故事，不是真实基本面
- 跟着最强资金的方向走，不做逆势判断
- 短线本质是"利用市场情绪的非理性"获利

### 5.3 大局观优先级
```
大盘趋势 > 情绪节点 > 题材强度 > 个股选择
不在对的环境做错的票，不在错的环境做对的模式
```
