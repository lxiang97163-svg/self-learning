# 05 操作模式详解

## 模式总体分类

```
操作模式
├── 强更强（进攻型）
│   ├── 三一模式（竞价阶段选股）
│   ├── 半路模式（盘中介入）
│   └── 9431梯队模式（板块选股）
│
└── 弱转强（反转型）
    ├── 先人指路（长上影线次日高开）
    ├── 断板反包（炸板次日高开）
    ├── 前排炸板低阶（被动炸板的日内机会）
    ├── 爆量高开转移制（爆量次日延续）
    ├── 连板弱转强（空头风标反核）
    ├── 一红定江山（横盘突破）
    └── 底部K线反转（举足/点灯/旭日等）
```

## 强更强模式

### 1. 三一模式（核心中的核心）

#### 1.1 三一的双层含义

**操作层三一**（选股指标）：
```
竞价阶段，目标票需同时满足以下3个第一：
  指标1: 集合竞价成交额第一（进额第一）
  指标2: 集合竞价换手率第一
  指标3: 集合竞价涨幅第一
```

**理念层三一**（交易哲学）：
```
第一: 强中选强，做范围内的第一名
唯一: 市场前期炒过、后续绕不开的标的（辨识度）
专一: 一击不重，不二击后排；大阳线不怕踏空
```

#### 1.2 三一比较范围

| 范围 | 定义 | 举例 |
|------|------|------|
| 全市场三一 | 全A股竞价最强的票 | 市场龙头、妖股 |
| 300三一 | 沪深300成份股里最强 | 大盘白马龙头 |
| 创业板三一 | 创业板内最强 | 高弹性中小票 |
| 题材三一 | 同一题材内最强 | 题材龙头 |
| 板块三一 | 同一行业板块内最强 | 行业龙头 |
| T队三一 | 同板数内最强 | 二板/三板/首板三一 |
| 票池三一 | 自选股票池内最强 | 个人关注池筛选 |

#### 1.3 三一的"半本剑谱"陷阱

**正确使用三一**：三个指标只是"前半本"，还需：

```
后半本（必须叠加验证）:
  1. 题材运行阶段（启动/上升期=可做；分歧/退潮=危险）
  2. 情绪节点判断（断板次日=禁止，启动期=猛干）
  3. 个股量价形态（盘子大小/前高压力/图形是否干净）

陷阱一（盘子太小）:
  "成也消和，败也消和"
  盘子<10亿时，换手极高→容易成三一
  但成交量达到流通盘30%+时，筹码极度混乱→不能参与
  判断方法：IF 流通市值<20亿 AND 当日换手>20% → 跳过

陷阱二（分歧期三一）:
  分歧期出现三一=抛压大，资金在博弈兑现
  方向与启动期相反
  判断方法：IF 龙头断板OR分歧明显 AND 三一出现 → 不参与

陷阱三（高位多次三一）:
  每次三一的买点价值递减
  第1次B点：满仓
  第2次B点：半仓
  第3次B点：轻仓
  第4次及以后：原则上不参与
```

#### 1.4 三一买点操作流程

```
Step 1 [9:15-9:20] 盯换手和进额排名
  → 筛选出进额和换手前5名的票

Step 2 [9:20-9:25] 确认涨幅+检验半本剑谱
  → 三个指标都靠前？题材主流？情绪节点对？
  → 盘子不太小？图形没大帽子？

Step 3 [9:25] 下单
  → 满足则9:25挂单，9:30开盘成交
  → 不满足则放弃，等下一个机会

Step 4 [9:30开盘] 确认盘口
  → 前3笔成交偏多→持有
  → 开盘下杀→等龙头回风后DC买点
```

#### 1.5 红三一 vs 绿三一

```
红三一: 
  - 当日整体涨势，筛选出进额最大+换手最高+涨幅最高的票
  - 典型：强势行情中的市场三一
  - 操作：正常追买

绿三一（空头风标反核）:
  - 当日整体下跌/分化，在下跌票中筛选出换手最高的
  - 典型：昨日跌停票今日反核竞价三一
  - 操作：逆向做多，赔率极高
  - 额外条件：情绪节点处于启动期 OR 该票历史地位高（妖股/核心龙头）
```

### 2. 半路模式

```
定义: 开盘后（9:30-14:30）中途发现量价异动，盘中介入

触发条件:
  - 盘中某段5分K突破当日高点（放量）
  - 该区间量能明显大于前几根5分K
  - 题材龙头同方向走强

买点:
  - 突破前高点的当根5分K收盘价附近
  - 或突破前高点后的第一次回踩（缩量）

不碰情形:
  - 追高超过5%以上（太贵）
  - 龙头已经高位大幅分化
  - 当日已经到了午后（14:00以后，时间窗口不够）
```

### 3. 9431板块选股模式

```
定义: 通过T队梯队判断题材强度

判断逻辑:
  - 九板→四板→三板→一板形成完整梯队
  - 梯队越整齐 = 题材持续性越强 = 板块买点越多

操作:
  - 在该板块内找三一或先人指路票
  - 梯队缺口（如有九板但无四板）= 题材可能分化

快速判断方法:
  - 同花顺660（个股复盘）按连板数排序
  - 找当日板块内各连板数的代表票，观察梯队完整度
```

## 弱转强模式

### 4. 先人指路模式

```
定义: 昨日上影线（触板未封/高开低走）→ 次日带量高开继续上涨

触发条件:
  昨日：出现长上影线（股价触及涨停但未封住）
  今日：集合竞价高开 OR 低开后极拉（低开极拉视为修复）
  额外：所在题材保持主流，龙头状态正常

买点:
  - 开盘即买（高确定性时）
  - 开盘下杀后，龙头回风时DC买入（确定性更高）
  - 突破5分K第一根高点（最稳但赔率减少）

止损:
  - 开盘下杀且无回风迹象 → 放弃
  - 题材龙头同时走弱 → 止损

多B点体系:
  第1 B点（先人指路当日开盘）: 最佳买点，赔率最高
  第2 B点（次日若再高开）: 好买点，但需控仓
  第3 B点及以后: 极轻仓或放弃，赔率已低
```

### 5. 断板反包模式

```
定义: 昨日连板中途断板 → 次日集合竞价高开重新确认

区别于先人指路：断板=完全开板跌下来；先人指路=上影但未完全开板

触发条件:
  昨日：连板票断板（涨停价被打开，且未回封）
  今日：集合竞价高开（通常需>2%）
  题材：该题材龙头状态良好（最好同日龙头没断板）

操作:
  - 开盘确认买入（或进价直接下单）
  - 盘中下杀但龙头回风→DC买入

禁止场景:
  - 所在题材整体退潮（多个票同时断板）
  - 昨日断板时成交量爆量（主力出货断板）→ 不碰
```

### 6. 前排炸板低阶

```
定义: 被动原因（指数下跌/情绪冲击）导致的前排强势股临时炸板，日内低接

炸板六原则（完整版）:
  原则1（时间顺序）: 
    - 后炸板的比先炸板的强
    - 选最后炸板的那只，不选第一个
    - 但若先炸板的先回风，存在卡位效应
    
  原则2（封单大小）: 
    - 炸板前封单越大越好
    - 封单大=买盘充足=回风力度强
    
  原则3（主动vs被动）:
    - 被动炸板（指数/情绪原因）→ 可参与低阶
    - 主动炸板（主力出货/无外力）→ 不碰
    
  原则4（放量速度）:
    - 炸板后放量越慢越好
    - 光速放量=筹码混乱=回风困难
    - 技巧：比较炸板那分钟的量柱 vs 封板时的量柱，炸板量<封板量=较强
    
  原则5（盘口变化）:
    - 炸板后主力净额方向
    - 持续向上=主力在接货=可跟
    - 急速下行=主力出货=不碰
    
  原则6（回风时间/力度）:
    - 回风越快越好
    - 回风封单越大越好
    - 示例：数据港10:53分回风后封单8.6亿=极强

操作买点:
  - 炸板后下探至支撑位
  - 主力净额开始向上
  - 若满足1/2/3原则则买入
```

### 7. 爆量高开转移制

```
定义: 股票某日爆量（成交量创近期新高）→ 次日高开继续上攻

成为妖股/龙头的路径:
  普通票 → 经历一次爆量高开转移制 → 市场龙头候选
  市场龙头 → 经历两次爆量高开转移制 → 正式成为"妖股"（有资金记忆）

妖股判断标准:
  完成两次爆量高开转移制 → 称为妖股
  特征：资金有记忆，容易产生二波行情

操作买点:
  第N次爆量高开（N=1或2）时:
  - 如果是第一次：满仓介入，最优买点
  - 如果是第二次：半仓，确认节点
  - 第三次以后：轻仓或不参与（买点靠后，赔率递减）

配合三一使用:
  IF 爆量高开 AND 进价三一 → 双重确认，最强买点
```

### 8. 一红定江山模式

```
定义: 长期横盘整理后，突破前115个交易日最高价，跳空高开

动态选股条件（系统筛选）:
  - 昨日开盘价 <= 前115个交易日的最高价
  - 今日开盘价 > 前115个交易日的最高价

逻辑:
  - 115个交易日 ≈ 半年，超过半年的套牢盘突破
  - 跳空突破=能量积累充分，向上空间巨大
  - 类似价值重估，但短线操作

买点:
  - 突破当日进价直接买（不等回调）
  - 收盘价需站在115日高点之上才有效

配合条件:
  - 必须有量（放量突破，不能无量假突破）
  - 题材配合（有故事支撑）
```

### 9. 底部K线反转信号

```
四种底部信号线（高阶456内容）:
  
  举足线: 下跌后出现底部小阳线，类似走路举脚的形态
  点灯线: 底部出现下影线明显的K线，像蜡烛点亮
  出日线: 底部出现缺口向上跳开，像太阳出现
  旭日线: 底部大阳线确认反转，但需要量能配合

预祝复堂线:
  - 形态：阴线被阳线反包（放量）
  - 需次日高开确认，高开才有效

预祝晴天线:
  - 形态：阴线被大阳线反包（更大量）
  - 信号更强，但同样需要高开确认
  
导星反转:
  - 顶部出现十字星（量智希）
  - 次日选择方向：高开→继续看多；低开→趋势反转

使用规则:
  IF 底部K线信号出现 AND 次日高开确认:
    → 有效，可买入
  IF 底部K线信号出现 AND 次日低开:
    → 信号失效，不操作
```

## 便宜老大模式（特殊场景）

```
定义: 题材内有强势大哥带动下，找题材内价格便宜的小弟

触发条件:
  1. 同题材龙头（大哥）出现四位数封单（盘口出现大单）
  2. 同题材内有价格明显偏低的个股
  3. 大哥强势上涨中

操作:
  - 在大哥出现四位数盘口时，同时买入题材内的便宜小弟
  - 看大哥的量能和封单来判断小弟能涨多少
  - 小弟通常跟涨1-3%，但涨停可能性也存在
```
