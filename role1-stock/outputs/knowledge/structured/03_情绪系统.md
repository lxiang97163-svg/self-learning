# 03 情绪系统

## 关键区分：情绪周期 vs 情绪节点

| 概念 | 定义 | 可操作性 |
|------|------|---------|
| **情绪周期** | 市场情绪的宏观描述（牛/熊/震荡） | ❌ 无法精准指导操作 |
| **情绪节点** | 题材/个股情绪的微观状态转换点 | ✅ 直接指导买卖时机 |

**核心原则**：操作看情绪节点，不看情绪周期。

---

## 1. 七个情绪节点定义

```
启动 → 发酵 → 高潮 → 分歧 → 分化 → 退潮 → 混沌 → [下一轮启动]
```

| 节点 | 市场特征 | 对应操作 | 情绪分 |
|------|----------|----------|--------|
| **启动** | 题材第一日出现多个一字板，资金快速聚集 | 猛开仓，给满分 | 3分 |
| **发酵** | 题材龙头连续封板，T队梯队整齐扩大 | 加仓，持股 | 2-3分 |
| **高潮** | 最高板创新高，封单达到历史最大，全市场追捧 | 轻仓，准备减仓 | 1-2分 |
| **分歧** | 龙头高开低走，封单减少，但未断板 | 控仓，做T不做底仓 | 1分 |
| **分化** | 部分个股开始大幅回撤，资金开始切换 | 减仓，留观 | 0-1分 |
| **退潮** | 龙头断板，整体题材大幅回调 | 空仓！ | 0分 |
| **混沌** | 无明确主线，资金电风扇轮动 | 空仓或极轻仓试探 | 0分 |

## 2. 情绪风标：读取情绪的工具

### 2.1 多头风标（做多情绪代表）

选择标准（优先级排序）：
1. **最高板股** — 当日连板数最多的票，是整个市场情绪的晴雨表
2. **题材老大** — 主流题材内最强的龙头
3. **券商手板** — 游资团队的旗手股（透露资金真实意图）
4. **最大封单** — 当日封单金额最大的票

**规则**：
```
IF 多头风标高开封板 → 情绪正常，按计划操作
IF 多头风标高开后断板 → 情绪即刻归零，立即减仓
IF 多头风标持续走强至收盘 → 次日情绪大概率延续
```

### 2.2 空头风标（做空情绪代表）

选择标准：
1. **最高板断板** — 连板最高的票当日未封住
2. **当日大面** — 当日跌幅最大的题材代表股
3. **最大跌停封单** — 跌停封单最大的票（主力出货信号）

**规则**：
```
IF 空头风标出现在情绪高潮期 → 高概率顶部，减仓信号
IF 空头风标出现在低位（混沌后）→ 可能是下一轮启动的前奏，空头风标反核买点
```

### 2.3 空头风标反核（弱转强买点）

```
触发条件:
  - 某票成为空头风标（跌停/断板）
  - 次日竞价该票出现集合竞价三一（进额/换手/涨幅排名靠前）
  - 情绪节点处于启动或发酵期
  
操作:
  → 9:25分看进价三一确认
  → 满足则9:30开盘买入（绿三一买点）
```

## 3. 竞价阶段情绪判断时间线

### 3.1 三阶段观察框架

```
9:15 → 9:20  [可挂可撤]
观察：封单是否在持续增加？
  增加 → 抢筹激烈，情绪偏多
  减少 → 撤单增加，情绪偏弱（凹槽卖点）

9:20 → 9:25  [只挂不撤]
观察：卖单是否在增加？
  卖单不增 → 筹码稳定，可买入
  卖单大增 → 抛压出现，谨慎或放弃
  
9:24 → 9:25  [最后1分钟]
观察：成交量是否急速拉升？
  急速拉升 → 尾盘抢筹激烈=最强做多信号
  平稳 → 正常竞价，参考其他指标
```

### 3.2 集合竞价分时图四象限解读

```
          上方（未成交挂单）
          红色=买单排队（抢筹）
          绿色=卖单排队（抛压）
          
          下方（已成交）
          红绿代表成交量分布
          趋势比颜色更重要
```

**四种典型形态**：

| 形态 | 特征 | 含义 |
|------|------|------|
| 上红大+下红小 | 买单多，成交少 | 一致做多，强势 |
| 上绿大+下绿大 | 卖单多，成交多 | 抛压极大，跌停候选 |
| 上红→缩→红（凹槽） | 9:20后先缩后扩 | 主力试盘后重新加仓=最佳买点 |
| 上绿→缩（尾盘） | 绿色减少，成交激增 | 跌停板被撬，买机会 |

## 4. 情绪打分与仓位对照表

### 4.1 情绪节点→情绪分值

| 情绪状态 | 触发信号 | 情绪分 | 最大允许仓位层数 |
|----------|----------|--------|----------------|
| 启动期 | 主流题材首日多个一字板 | 3分 | 3层 |
| 上升期 | 龙头持续封板，T队扩大 | 2分 | 2层 |
| 高潮期 | 最高板达到历史高位 | 1分 | 1层 |
| 分歧期 | 龙头开板但未断 | 1分 | 1层（做T不持仓）|
| 退潮期 | 龙头断板 | 0分 | 0层（空仓）|
| 断板次日 | 龙头断板后第一个交易日 | 0分 | 0层（必须空仓）|
| 混沌期 | 无主线，电风扇轮动 | 0分 | 0层 |

### 4.2 情绪节点判断速查

```python
# 伪代码：情绪节点判断逻辑
def judge_emotion_node(market_data):
    if market_data.highest_board.封板成功 and market_data.t队梯队.整齐:
        return "启动/发酵"
    elif market_data.highest_board.封板成功 and market_data.board_count >= 历史高位:
        return "高潮"
    elif market_data.highest_board.开板 and NOT market_data.highest_board.断板:
        return "分歧"
    elif market_data.highest_board.断板:
        return "退潮"  # 情绪=0，立即降仓
    elif 无明确最高板:
        return "混沌"
```

## 5. 情绪风标票池维护规则

- 每月更新一次情绪风标票池
- 删除标准：该票不再能引领市场情绪（成交萎缩/题材退潮）
- 新增标准：近期走出强势行情、辨识度高、游资重点关注的票
- 票池大小：通常10-20只，覆盖各主流题材各1-2只

**票池分类**：
1. AI/算力类代表票
2. 机器人/新能源类代表票
3. 传统题材（地产/消费/银行）护盘类票
4. 当前最热题材的龙头1-2只

## 6. 情绪判断反常识规则

```
RULE 1: 大阳线当日不一定能买入
  大阳线当日 = 已经完成了快速上涨，追高赔率低
  大阳线次日高开 = 更好的买点（右侧确认+情绪延续）

RULE 2: 情绪高潮时要减仓不是加仓
  高潮期特征：全市场都在讨论的票
  此时加仓 = 最后接盘

RULE 3: 指数大涨不等于可以买所有票
  指数涨由护盘股（银行/券商/A50）拉动 ≠ 情绪真实启动
  真实启动：题材股齐涨 + 封板数量增加 + 换手活跃

RULE 4: 断板次日必须空仓
  断板=龙头崩盘=情绪清零
  断板次日情绪=0分，无论个股多强，都不碰
```
