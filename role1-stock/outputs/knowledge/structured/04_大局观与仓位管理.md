# 04 大局观与仓位管理

## 核心框架：4:3:2:1仓位体系

**总仓位 = 指数(4) + 情绪(3) + 题材(2) + 个股(1) = 最多10层仓**

每层仓 ≈ 总资金的10%。满分=满仓，低分=轻仓或空仓。

---

## 1. 指数大局观（4分维度）

### 1.1 指数关键点位确定方法

**选点逻辑**（从大到小看）：
```
月线级别 → 周线级别 → 日线级别 → 5分线级别

核心参考点来源：
1. 历史高低点（前期重要支撑/压力）
2. 重大政策日的收盘价（如828券商晚宴=3089点）
3. 均线（MA20/MA60为主，不用MA5）
4. 前期成交量密集区（套牢盘区域）
```

### 1.2 指数打分规则

| 指数状态 | 打分 | 描述 |
|---------|------|------|
| 跳空高开+站上关键点位 | 4分 | 供给盘开盘，可主动进攻 |
| 高开但未突破关键点位 | 3分 | 偏多，正常操作 |
| 平开/小幅低开，支撑良好 | 2分 | 中性，轻仓观察 |
| 低开在关键点位之下 | 1分 | 偏弱，不操作或仅防守 |
| 跳空低开+空方仙人指路 | 0分 | 禁止开仓，空仓 |
| 破位（如破3031/2984等关键点） | -1分（清仓信号） | 周线级别调整风险 |

### 1.3 指数操作规则（盘前预案）

```
每日必做：在5分K图上标注当日关键点位

IF 高开站上关键点位:
    → 指数得4分，大举进攻
    
IF 低开但在关键支撑上方:
    → 指数得1-2分，等盘中机会
    
IF 低开破关键支撑:
    → 指数得0分，禁止所有买入操作
    → 已有持仓：减仓至1层以下

IF 出现空方仙人指路（跳空低开K线）:
    → 必须空仓，等待指数收复信号
```

### 1.4 指数K线信号速查

| K线组合 | 名称 | 含义 | 次日操作 |
|---------|------|------|---------|
| 阴线被放量阳线反包 | 预祝晴天 | 需高开确认 | IF高开→做多；IF低开→不确认，继续观察 |
| 阴线被缩量阳线反包 | 预祝复堂 | 偏弱，需高开确认 | 同上，可信度稍低 |
| 量智希十字星 | 变盘信号 | 下一日选择方向 | 高开→做多；低开→做空/空仓 |
| 跳空高开后单日极杀 | 高开低走确认 | 诱多陷阱 | 减仓 |
| 爆量大阴线 | 出货信号 | 主力兑现 | 次日不追，等缩量反包 |

## 2. 持仓管理四大原则

### 2.1 持仓低开的三条处理思路

```
思路1（持仓低开，龙头和指数均正常）:
  → 持仓不动，等待回风
  → 如有加仓机会（DC/炸板低阶）可轻加
  
思路2（持仓低开，题材龙头也走弱）:
  → 减仓1/3，留底仓观察
  → 若龙头持续走弱 → 再减至轻仓
  
思路3（持仓低开，指数+情绪双弱）:
  → 9:30开盘即清仓
  → 宁可斩掉浮亏，下次再找机会
```

### 2.2 倒金字塔加仓禁令

```
禁止规则：越涨越加（高位金字塔加仓）
  ❌ 第1次买入1层 → 涨5%后再加2层 → 再涨后再加3层
  
正确规则：
  ✅ 第1笔（模式B点） → 给足仓位（如3层）
  ✅ 第2笔（更好B点出现）→ 补仓但不超过第1笔
  ✅ 有利润电后做T：买T仓 ≤ 当前底仓，且当日必须卖出T仓
```

### 2.3 日内做T规则（等仓原则）

```
基本规则：买入≤当前底仓，当日必须等仓卖出

示例：
  底仓：3层
  做T买入：最多3层
  当日收盘前：必须卖出T仓（回到3层底仓）
  
❌ 错误：做T后持仓从3层变5层（不允许）
✅ 正确：做T后日内出掉T仓，底仓始终=3层
```

### 2.4 止损/离场规则

```
短线票止损规则:
  - 开盘不满足模式 → 开盘即减仓
  - 盘中破5分K关键支撑且无龙头配合 → 止损
  - 持仓变成最差（非主线/非龙头/图形破坏）→ 切换
  
趋势票止损规则:
  - 建仓次日未能做出利润电 → 认错出局（无论如何）
  - 破关键支撑（MA5/MA20/前期低点）→ 止损
  - 发射失败=火箭墜毀，不等待，立刻出局
```

## 3. 仓位分配实操

### 3.1 标准仓位计算公式

```
当日最大仓位 = 指数分(0-4) + 情绪分(0-3) + 题材分(0-2) + 个股分(0-1)

示例A（理想环境）:
  指数4 + 情绪3 + 题材2 + 个股1 = 10层 = 满仓

示例B（震荡环境）:
  指数2 + 情绪1 + 题材1 + 个股1 = 5层 = 半仓

示例C（弱市）:
  指数1 + 情绪0 + 题材1 + 个股0 = 2层 = 极轻仓观察

示例D（退潮期）:
  任意维度情绪=0 → 仓位≤2层（防御模式）
```

### 3.2 单票仓位上限

```
普通情绪环境（启动/上升期）:
  - 短线票单票上限：3-4层
  - 趋势票（环境极好时）：6-8层

情绪高潮/分歧期:
  - 短线票单票上限：1-2层（日内做T为主）
  - 趋势票：2-3层，有利润电后轻仓持

退潮期:
  - 任何票：最多1层（仅用于探路）
  - 建议：空仓等待
```

## 4. 预案制作方法

### 4.1 盘前必做事项（每日固定流程）

```
1. 看指数（5分K）:
   - 标注当日关键点位
   - 判断今日指数得分
   - 写下：指数高开X点以上可操作 / 低开X点以下不操作

2. 看情绪风标:
   - 今日情绪代表票是什么？
   - 预计风标高开 or 低开？
   - 情绪得分填入

3. 看题材:
   - 昨日主流题材是什么？今日是否延续？
   - 9:25分竞价最强题材是什么（实时判断）？

4. 确定票池:
   - 今日优先做什么模式的票？
   - 写出3-5只备选票及对应买点价格

5. 写出IF-THEN预案:
   - IF 指数高开3031 → THEN 在XX价格买XX
   - IF 指数低开 → THEN 空仓观察
```

### 4.2 中国金龙作为外盘参考

```
中国金龙指数（KraneShares CSI China Internet ETF）:
  - 代表海外市场对A股中概股的看法
  - 夜盘看中国金龙走势 → 预判次日A股科技/互联网板块方向

使用规则:
  IF 中国金龙夜盘大涨 → 次日科技/AI类题材预期偏多
  IF 中国金龙夜盘大跌 → 次日科技类题材预期偏空，指数可能受压
  IF 中国金龙横盘 → 外盘无明确指引，看A股自身情绪
```

## 5. 大局观的多维度检验

### 5.1 三层验证体系

```
Layer 1（宏观）: 指数是否在大趋势支撑位以上？
Layer 2（中观）: 题材是否有明确主线？连续3日以上？
Layer 3（微观）: 目标个股是否在题材内排名靠前？

三层同向 → 高确定性买点
两层同向 → 中等确定性，轻仓
只有一层 → 低确定性，极轻仓或不做
```

### 5.2 护盘力量识别

```
主要护盘板块（指数权重股）:
  - 银行（招行/工行）
  - 券商（中信/华泰）
  - A50（上证50成份股）
  - 中字头（中船/中传等）

规则:
  IF 护盘板块早盘即异动 → 市场意图维稳，大概率无大跌
  IF 护盘板块提前动手（10点前拉升）→ 后续效果有限（吃力不讨好）
  IF 护盘板块午后才介入+砸出恐慌量→ 最佳护盘效果，次日反弹概率高
  
  最强空头板块:
  - 茅台（白酒/消费）大幅下跌=市场无信心信号
  - 上证50重要成份股齐杀=系统性风险
```
