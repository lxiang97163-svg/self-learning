# 02 开盘八法

## 核心原理

**开盘=当日多空力量第一次公开博弈的结果**。开盘位置直接反映了主力资金对当日预期的判断，比任何技术指标都更直接。

八法按昨收价偏移程度分为三档（三高/二高/一高/平移/一低/二低/三低），共8种形态，每种均有对应的操作策略。

---

## 八法速查表

| 形态 | 开盘位置特征 | 情绪含义 | 操作策略 |
|------|------------|---------|---------|
| **三高** | 高开+竞价强+封单厚 | 市场极度一致看多 | 强势追买，但注意高位分歧风险 |
| **二高** | 高开+竞价尚可 | 多头主导，有一定共识 | 正常买入，控制仓位 |
| **一高** | 小幅高开，竞价平淡 | 偏多，但共识一般 | 轻仓参与，观察盘中确认 |
| **平移上** | 平开后快速拉升 | 开盘蓄势，资金盘中发力 | 盘中追入，以5分K站上均线为信号 |
| **平移下** | 平开后快速下杀 | 弱势整理或出货 | 不操作，等待止跌信号 |
| **一低** | 小幅低开，支撑良好 | 偏弱但题材仍在 | 等龙头稳住后再参与 |
| **二低** | 低开+有一定抛压 | 情绪走弱，分歧出现 | 谨慎，宁可踏空 |
| **三低** | 大幅低开+抛压明显 | 情绪崩溃或系统性下跌 | 空仓，不参与 |

---

## 三高开盘详解

### 定义
- 集合竞价涨幅高（通常>3%）
- 封单量大（通常>5000万，市场情绪高时可达数亿）
- 换手排名靠前，进额排名靠前

### 操作规则
```
IF 三高开盘 AND 题材主流 AND 情绪启动节点:
    → 开盘第1-2分钟内买入（不等回调）
    → 仓位：满足三一模式则可给足
    
IF 三高开盘 BUT 题材已经高潮分化期:
    → 轻仓参与，日内做T为主
    → 当日超预期则追，不超预期则出
```

### 陷阱识别（三高的失效场景）
- 盘子太小（<10亿流通市值）导致的虚假三高 → 成也消和，败也消和
- 情绪分化期或退潮期的三高 → 抛压大，分歧猛烈，反向信号
- 高位连续三高后的三高 → 通常是顶部，需要减仓而非加仓

---

## 一高开盘详解（先人指路模式入口）

### 定义
- 集合竞价小幅高开（1%-5%）
- 对应昨日炸板（涨停板被打开）的次日高开

### 操作规则（先人指路）
```
昨日条件:
  - 股票出现长上影线（触板但未封死）
  - 或尾盘强反弹后收在当日高位
  
今日触发:
  IF 集合竞价高开（低开极拉也算修复）
  AND 所在题材龙头状态正常
  AND 竞价进额/换手靠前:
    → 9:30开盘即买入（不追，有5分K突破再追）
    → 若开盘下杀：等龙头回风后DC买入
```

---

## 平移开盘详解

### 平移上（半路模式入口）

**特征**：平开后5-15分钟内资金介入，量能异动，股价快速拉升

**操作规则**：
```
IF 平开 AND 5分K出现明显放量+价格突破开盘高点:
    → 突破点买入（视为盘中三一信号）
    → 题材龙头同步走强则确定性更高
```

### 平移下（弱转强候选）

**特征**：平开后缓慢下行，无量萎缩

**操作规则**：
```
IF 平移下 AND 缩量触及支撑（MA5/MA10/前期高点）:
    AND 大单净额向上
    → 观察止跌信号（分时底背离）
    → 不主动买入，等反弹确认
```

---

## 低开系列详解

### 一低：前排炸板低阶入场点

**发生背景**：指数低开 / 情绪不及预期 → 前排强势股被动炸板

**操作规则（炸板六原则前3条）**：
```
条件1（时间）: 炸板时间越晚越强，不选第一个炸的
条件2（封单）: 炸板前封单越大越好（买盘充沛）
条件3（性质）: 被动炸板（因指数/情绪）> 主动炸板（主力出货）

IF 满足上述3条 AND 炸板后放量缓慢（非光速放量）:
    → 炸板后下探至支撑位买入
    → 看回风速度和封单恢复情况
```

### 二低/三低：空仓等待

```
IF 开盘二低 OR 三低:
    → 禁止买入
    → 观察指数关键支撑是否守住
    → 等待情绪节点由退潮转为启动的信号
```

---

## 竞价时间线操作规则

```
9:15 - 9:20  挂单可撤单阶段
  → 观察主力初始意图，可小仓试探

9:20 - 9:25  只能挂不能撤（凹槽模式观察点）
  → 此阶段撤单=凹槽信号（主力减仓）
  → 封单从大变小再变大=最佳抢筹机会

9:25 - 9:30  禁止撤买单（技术原因，避免委托丢失）
  → 9:25分进额急速拉升=尾盘抢筹激烈=强烈做多信号

9:30  正式开盘
  → 前3笔成交回报判断盘口一致性
  → 买方主导→继续持有/加仓
  → 卖方主导→等待回测再入场
```

---

## 开盘八法与情绪的联动规则

```
情绪启动期 + 三高/二高开盘 = 最强买点，满仓进攻
情绪上升期 + 一高/平移上 = 正常买点，控仓
情绪高潮期 + 三高开盘 = 警惕，日内做T
情绪分歧期 + 任何开盘 = 轻仓或不操作
情绪退潮期 + 任何低开 = 空仓，等下一个周期
```
